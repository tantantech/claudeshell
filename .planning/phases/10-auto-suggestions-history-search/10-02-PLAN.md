    - On right-arrow when cursor NOT at end of line: normal behavior, no acceptance (per research Pitfall 2)
    - On escape or Ctrl+C when suggestion active: clearGhost, suggestion cleared (per D-11)

  <action>
    Create `src/suggestions/keypress.ts` per D-10:

    1. Export `createKeypressHandler(rl: readline.Interface, history: readonly string[], filters: readonly RegExp[], debounceMs: number)`:
       - Returns a keypress handler function `(str: string, key: readline.Key) => void`
       - Module state: `let activeSuggestion: string | null = null` and `let debounceTimer: ReturnType<typeof setTimeout> | undefined`

    2. Handler logic (fires on every `process.stdin` 'keypress' event):
       a. **Always first**: call `clearGhost()` (per research Pitfall 1 -- ghost text persists after simple char append)
       b. **Right-arrow acceptance** (per D-08): if `key.name === 'right' && activeSuggestion !== null && rl.cursor === rl.line.length`, call `rl.write(activeSuggestion)`, set `activeSuggestion = null`, return
       c. **Escape/Ctrl+C** (per D-11): if `key.name === 'escape' || (key.ctrl && key.name === 'c')`, set `activeSuggestion = null`, clear debounce timer, return
       d. **Tab** (per research): if `key.name === 'tab'`, set `activeSuggestion = null`, clear debounce timer, return
       e. **Return/Enter**: set `activeSuggestion = null`, clear debounce timer, return
       f. **Default -- schedule suggestion update**: clear existing debounce timer, set new timer at `debounceMs`:
          - In timer callback: read `rl.line` (already updated per research Pattern 1)
          - Call `findSuggestion(rl.line, history, filters)`
          - If match found: compute `suffix = match.slice(rl.line.length)`, call `renderGhost(suffix)`, set `activeSuggestion = suffix`
          - If no match: set `activeSuggestion = null`

    3. Type the `key` parameter using `readline.Key` (import from `node:readline`).

    Create `tests/suggestions/keypress.test.ts`:
    - Mock rl (with line, cursor, write method), mock process.stdin
    - Test: printable key triggers debounced findSuggestion + renderGhost
    - Test: right-arrow with active suggestion calls rl.write with suffix
    - Test: right-arrow with cursor mid-line does NOT accept
    - Test: escape clears suggestion
    - Test: tab clears suggestion without scheduling new one
