# assets

## Purpose
Visual assets for README and marketing — demo GIF, VHS recording tape, and logo.

## Key Files

| File | Description |
|------|-------------|
| `demo.gif` | Animated demo shown in README — generated by VHS |
| `demo.tape` | VHS tape file that generates `demo.gif` |
| `demo-init.sh` | Bash init script for VHS recording — simulates nesh prompt and commands |
| `logo.svg` | Nesh logo used in README header |
| `demo.svg` | SVG version of demo (unused) |

## For AI Agents

### Working In This Directory
- To regenerate the demo: `vhs assets/demo.tape`
- `demo-init.sh` defines shell functions that simulate nesh output for recording
- The tape uses `demo-init.sh` as a bash rcfile, then `clear` to hide setup
- VHS `Hide`/`Show` does NOT work reliably — use `clear` instead
- Keep demo under 20 seconds for GitHub README autoplay

### Common Patterns
- VHS `Set Shell "bash"` — custom shells not supported
- Simulated output via bash function overrides (`ls`, `a`, `nesh`, `git`)
- ANSI colors via `\033[...m` escape codes in printf
