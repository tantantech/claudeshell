# ui

## Purpose
shadcn UI primitives — reusable, styled components generated by the shadcn CLI.

## Key Files

| File | Description |
|------|-------------|
| `badge.tsx` | Badge/tag component |
| `button.tsx` | Button with variants (default, outline, ghost, etc.) |
| `card.tsx` | Card container with header, content, footer |
| `separator.tsx` | Visual separator/divider |
| `switch.tsx` | Toggle switch component |

## For AI Agents

### Working In This Directory
- These are shadcn components — add new ones via `npx shadcn add <component>`
- Uses `class-variance-authority` for variant styling
- Uses `tailwind-merge` + `clsx` via `lib/utils.ts` for class composition
- Do not heavily customize — prefer wrapping in landing-specific components
